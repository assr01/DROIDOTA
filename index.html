How to Build

  • Click on SignIn to continue
  • Only google accounts allowed

  • Design your page using inbuild Editor
  • Paste your HTML code if designed earlier

  • Click on Render
  • Change the code to suit your needs

  • Voila, all set to Build
  • Hit Build to Get customised Apk Simple!
